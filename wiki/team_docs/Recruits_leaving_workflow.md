# 入职离职流

## 入职流程
* 创建邮件账号，id及alias。
* 设置邮件项目组。
* 设置Dropbox分享组。
* 开通置Trello账号及项目组。
* 加入iMessage、微信群组。
* 提交 public key 开通测试服务器帐号。
* 提交 public key 开通 panda-git (例如：wiki) 帐号。
* 欢迎邮件。


## 离职流程
* 退还使用的设备。
* 挂起邮件账号。
* 关闭代码库、数据库权限。
* 删除服务器key：线上服务器、内网测试服务器
* 关闭第三方账号：Dropbox、Trello、Apple Developer、iTunes Connect、Facebook、Testflight、Google Docs分享权限
* 删除iMessage、微信群组。
